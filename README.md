# Trade Pilot AI

> An advanced AI-powered application to guide aspiring exporters through every step of their journey, from product selection and profit calculation to finding buyers and navigating compliance.

Trade Pilot AI is a sophisticated, all-in-one web application designed to be the ultimate co-pilot for new and aspiring exporters. It leverages the power of the Google Gemini API to provide real-time data analysis, strategic guidance, and operational tools, simplifying the complexities of international trade.

## ‚ú® Features

- **AI-Powered Guidance:** Utilizes the Google Gemini API for market intelligence, buyer/supplier discovery, and content generation.
- **Comprehensive Modules:** A full suite of tools covering every aspect of the export lifecycle.
- **Dynamic & Responsive UI:** Built with React and Tailwind CSS for a modern, mobile-friendly experience.
- **Data-Driven Insights:** Tools for risk analysis, profit calculation, and market trend analysis.
- **Highly Customizable:** Features extensive settings for theming, language, currency, and navigation layout.
- **Simulated Experience:** Provides a safe, interactive environment to learn export processes, from planning to booking a consignment.

## üöÄ Getting Started

This is a self-contained web application that runs entirely in the browser.

1.  **Login:** The application starts with a simulated login screen. Enter any valid email and password to proceed.
2.  **Dashboard:** After logging in, you'll land on the main dashboard, which provides quick links to all the major modules.
3.  **Navigation:** Use the collapsible sidebar on the left to navigate between the different tools and features.
4.  **Explore:** Each module is designed to be intuitive. Start by exploring the **Study Guide** to understand the fundamentals, then use tools like the **Buyer Finder** or **Profit Calculator** to put the concepts into practice.
